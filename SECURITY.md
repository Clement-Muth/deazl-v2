# Security Policy

## Supported Versions

Seule la dernière version en production est maintenue.

## Reporting a Vulnerability

Pour signaler une vulnérabilité de sécurité, **ne pas ouvrir d'issue publique**.

Envoyer un email à : **security@deazl.fr**

Nous nous engageons à répondre sous 48h et à publier un correctif dans les 7 jours pour les vulnérabilités critiques.
