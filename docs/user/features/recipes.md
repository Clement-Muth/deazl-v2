# Recettes

Ta bibliothèque de recettes, c'est la base de tout dans Deazl. Chaque recette que tu ajoutes devient une brique que tu peux planifier et dont tu peux générer la liste de courses automatiquement.

---

## Ajouter une recette

Appuie sur **+** dans l'onglet **Recettes** et remplis les informations :

| Champ | Ce que ça change |
|---|---|
| **Nom** | Retrouve ta recette facilement |
| **Ingrédients** | Ce qui sera ajouté à ta liste de courses |
| **Portions** | La base pour adapter les quantités à ton foyer |
| **Temps** | Aide à planifier selon le soir disponible |
| **Étapes** | Ta recette complète, accessible pendant la préparation |

{% hint style="success" %}
**La magie des portions** — Si ta recette est pour 4 personnes et que tu es 2, Deazl divise automatiquement tous les ingrédients par 2 dans ta liste. Tu n'as rien à recalculer.
{% endhint %}

---

## Retrouver une recette

Utilise la barre de recherche pour trouver une recette par son nom ou par un ingrédient. Tes recettes préférées peuvent être marquées pour les retrouver en priorité.

---

## Partager une recette

Chaque recette génère un lien partageable. La personne qui le reçoit peut voir la recette complète sur le web — sans avoir besoin d'installer Deazl.

Pratique pour partager avec un proche ou retrouver une recette depuis ton ordinateur.

---

## Données nutritionnelles

Quand Deazl reconnaît un ingrédient dans la base Open Food Facts, il affiche automatiquement les informations nutritionnelles de la recette — calories, protéines, lipides. Sans que tu n'aies à saisir quoi que ce soit.

{% hint style="info" %}
Open Food Facts est une base de données alimentaires collaborative et ouverte. Plus les produits sont connus, plus les données sont complètes.
{% endhint %}
