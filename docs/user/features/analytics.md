# Comparaison des prix

La plupart des apps de courses te montrent des prix génériques tirés d'internet. Deazl fait différemment : il apprend les **prix réels** dans **tes magasins**, à partir de ce que toi et les autres utilisateurs scannez.

---

## D'où viennent les prix ?

---

## Lire la comparaison

Dans l'onglet **Comparaison**, tu vois pour chaque article de ta liste :

- Le **prix dans chacun de tes magasins**
- Le **magasin le moins cher** pour cet article
- L'**économie possible** si tu changes de magasin

---

## Où faire mes courses ?

Deazl peut calculer dans quel magasin faire l'ensemble de tes courses revient le moins cher — en tenant compte du panier complet, pas article par article.

C'est utile quand tu n'as pas envie de multiplier les arrêts mais que tu veux quand même optimiser ton budget.

{% hint style="success" %}
**Ça s'améliore avec le temps.** Les premières semaines, les comparaisons peuvent être limitées si tu n'as pas encore beaucoup scanné. Après un mois d'utilisation régulière, Deazl connaît précisément tes magasins et tes produits habituels.
{% endhint %}
