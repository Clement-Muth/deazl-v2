# Développement

## Prérequis

- Bun >= 1.x
- Node.js >= 20 (pour les outils qui en ont besoin)
- Compte Supabase

## Setup

```bash
git clone <repo>
cd deazl-v2
bun install
cp .env.example .env.local
bun run dev
```
