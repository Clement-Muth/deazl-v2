# Décisions d'architecture (ADR)

Les ADR (Architecture Decision Records) documentent les choix techniques significatifs : pourquoi on a choisi X plutôt que Y, les alternatives évaluées, et les compromis acceptés.

## Format

Chaque ADR suit ce format :
- **Contexte** — la situation qui a amené la décision
- **Décision** — ce qu'on a choisi
- **Alternatives considérées** — ce qu'on n'a pas retenu et pourquoi
- **Conséquences** — trade-offs acceptés
